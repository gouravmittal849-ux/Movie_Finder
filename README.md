# Movie Finder 🎬

## Description
Movie Finder is a web application that allows users to search for movies and series using the OMDb API. It displays movie details like title, year, and poster dynamically.

## Features
- Search movies in real time
- Filter by type (Movie / Series)
- Sort by year (ascending / descending)
- Loading indicator while fetching data
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- OMDb API

## API Used
https://www.omdbapi.com/

## How to Run
1. Clone the repository
2. Open index.html in your browser
